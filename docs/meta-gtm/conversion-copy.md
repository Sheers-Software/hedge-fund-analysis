## 6. Meta Compliance Do / Don't Cheat-Sheet

### Hard rules (never break)
- **No guaranteed or implied returns.** Never suggest the product makes money, grows wealth, or improves outcomes.
- **No "beat the market," "outperform," "get rich," "double your money," "10x," "winning picks," profit screenshots, or P&L imagery.**
- **No personalized financial advice.** Never "we'll tell you what to buy," "the best stock to buy now," or "our picks."
- **No urgency tied to financial outcome** ("buy before it's too late to profit").
- **Always frame as research / analysis / education / time-saving.**
- **Always preserve agency:** "you make the call," "you set the assumptions," "you decide."
- Include a research/education disclaimer on the landing page and in emails.

### Do
- "Research," "analysis," "investment memo," "fundamentals," "fair-value check," "valuation model," "saves you time," "for the price of a coffee."
- "Over- or undervalued" (a neutral analytical read, not a recommendation).
- Compare cost to a Bloomberg terminal (a *cost/tooling* claim, not a returns claim).
- Pose the user's own question back to them ("Is it overvalued right now?").

### Don't
- Don't promise accuracy, profit, or that the analysis is "right."
- Don't imply ApexAlpha replaces a licensed advisor.
- Don't use rocket emojis next to dollar figures or returns; keep ⚡/🤔 to *curiosity*, not gains.
- Don't show fake testimonials or earnings claims.

### Safe vs Risky phrases
| ✅ Safe | ❌ Risky (do NOT use) |
|---|---|
| "Hedge-fund-grade research" | "Hedge-fund-level returns" |
| "Is it over- or undervalued?" | "Is it about to pop?" |
| "Estimate fair value" | "Find the next 10x" |
| "Saves you hours of research" | "Makes you a better trader" (outcome claim) |
| "You make the call" | "We'll tell you what to buy" |
| "For the price of a coffee" | "Pays for itself in one trade" |
| "Read the thesis before you decide" | "Don't miss the next big winner" |
| "Research & education tool" | "Investment advice you can trust" |

## 7. Naming Convention + UTM Scheme

### Campaign / ad-set / ad naming
**Campaign:** `AA_[Objective]_[Audience]_[Funnel]_[YYYYMM]`
- Example: `AA_Traffic_DIY_TOF_202606`

**Ad set:** `[Audience]_[Targeting]_[Placement]`
- Example: `DIY_Lookalike1pct_ReelsFeed`
- Audience codes: `DIY` · `FOMO` · `VALUE` · `RT-VISIT` (R1) · `RT-FREE` (R2)

**Ad:** `[Angle]_[Format]_[HookID]_v[n]`
- Angle codes: DIY `A1`/`A2`/`A3`, FOMO `F1`/`F2`/`F3`, Value `V1`/`V2`/`V3`
- Format: `Static` · `Reel` · `Carousel`
- Example: `A1_Reel_HookSpreadsheet_v2`

### UTM scheme (append to all destination URLs)
```
?utm_source=meta
&utm_medium=paid_social
&utm_campaign={campaign_name}      // e.g. AA_Traffic_DIY_TOF_202606
&utm_content={ad_name}             // e.g. A1_Reel_HookSpreadsheet_v2
&utm_term={audience}_{funnel}      // e.g. DIY_TOF
```
**Example full URL:**
`https://apexalpha.app/?utm_source=meta&utm_medium=paid_social&utm_campaign=AA_Traffic_FOMO_TOF_202606&utm_content=F1_Reel_IsNVDAOvervalued_v1&utm_term=FOMO_TOF`

**Mapping to conversions:**
- Tag the **free signup** event with the landing `utm_campaign` + `utm_content` (capture on signup, store on the user record).
- Tag the **Pro upgrade** event with the *originating* signup UTMs (first-touch) so you can see which ad drove paying users, not just clicks.
- Use `utm_term` (`audience_funnel`) for fast pivot tables: which funnel stage and audience produce the cheapest signup vs the cheapest upgrade.

## 8. The First $50 Test

> Goal: cheapest path to *learning* which audience + angle converts a signup. Keep it to ~4 ads so each gets enough budget to read signal. Suggested split: ~$12–13/ad over 3–4 days, traffic/leads objective to free signup, broad-ish targeting + one interest stack.

**Launch these 4:**

2. **DIY MOF — "Spreadsheet weekend" story-led (Primary #3)**
   *Why:* core ICP, highest-volume audience, the clearest pain. This is your benchmark for blended CAC. Ad: `A1_Static_HookSpreadsheet_v1`.

3. **Value MOF — "Your DCF, always up to date" (Primary #1)**
   *Why:* highest intent and willingness to pay; likely best signup→Pro conversion even if CPC is higher. Tests quality of signup, not just quantity. Ad: `V1_Static_DCFUpkeep_v1`.

4. **DIY TOF — "Price of a coffee / Bloomberg contrast" (Primary #6)**

- **Cost per free signup** (efficiency) — expect FOMO TOF to win.
